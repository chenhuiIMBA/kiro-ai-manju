# 阶段 12：成片检查

> 工具：AI 多模态自检（截帧图片检查）+ 用户最终审核（音频）
> ⚠️ 画面和字幕由 AI 通过截帧自主检查，音频由用户在看成片时顺带确认

## 前置条件

- `06-output/ep{NN}-final.mp4` 存在
- 所有故事板关键帧、角色设定图可供对比

## 流程

### 步骤 0：视频截帧（AI 自检前置）

> 每段视频生成后立即截帧检查（不等到合成后）。合成后的成片再做一次全片截帧确认。

```bash
# 每段视频每 2 秒截一帧
ffmpeg -i ./05-videos/ep{NN}/seg1.mp4 -vf "fps=0.5" ./07-consistency-check/ep{NN}_seg1_%03d.jpg
ffmpeg -i ./05-videos/ep{NN}/seg2.mp4 -vf "fps=0.5" ./07-consistency-check/ep{NN}_seg2_%03d.jpg
ffmpeg -i ./05-videos/ep{NN}/seg3.mp4 -vf "fps=0.5" ./07-consistency-check/ep{NN}_seg3_%03d.jpg
ffmpeg -i ./05-videos/ep{NN}/seg4.mp4 -vf "fps=0.5" ./07-consistency-check/ep{NN}_seg4_%03d.jpg
```

AI 逐帧检查以下项目（通过多模态图片理解）：

### AI 自检项目（画面）
- [ ] 角色外观与设定图一致
- [ ] 画风与 Moodboard 一致
- [ ] 景别切换节奏舒适
- [ ] 无画面撕裂/闪烁/变形
- [ ] 色彩/光影与灯光哲学一致

#### 光影专项
- [ ] 主光方向在镜头间保持一致（非叙事动机变化时）
- [ ] 光比与场景情绪匹配，无随机漂移
- [ ] 色温在整段内无意外跳变
- [ ] 角色面部曝光一致（不突然过亮或过暗）
- [ ] 分离光有效（角色不从暗背景中消失）
- [ ] 灯光弧线与剧本灯光节拍对齐
- [ ] 特写中眼神光存在且方向一致
- [ ] 实用光源有画面内动机（如灯光哲学选择现实主义）
- [ ] 灯光母题在关键帧中可见

#### 字幕（AI 自检——通过截帧图片识别）
- [ ] 字幕文字与剧本对白一致（逐句对比截帧中的字幕文字和 ep{NN}-script.md 原文）
- [ ] 无形近字替换错误（如"各→何"）
- [ ] 字幕不遮挡关键画面

#### 声音（⚠️ 用户最终审核——AI 无法听音频）
- [ ] 配音清晰可辨，无含混吞字
- [ ] 角色声音区分度明显（A 和 B 闭眼能分辨）
- [ ] 🎤 各角色音色与 voice-ref.mp3 一致
- [ ] 情感表现符合剧本标注（愤怒够怒、悲伤不假）
- [ ] 语速节奏自然，与画面情绪同步
- [ ] 环境音效符合场景
- [ ] 音量无突变/爆音
- [ ] 对白无缺失（每句剧本对白都在成片中出现）

#### AI 生成专项
- [ ] 无手指/肢体变形（多指、扭曲、消失）
- [ ] 面部身份一致（同段内角色面部无漂移变脸）
- [ ] 无物理逻辑错误（衣服穿透、物品凭空消失、比例突变）
- [ ] 口型与配音同步程度可接受
- [ ] 无字幕/文字乱码（AI 生成的文字经常是无意义符号）
- [ ] 无非自然闪烁/帧间色彩跳变

#### 叙事（AI 自检——对照剧本）
- [ ] 3 秒钩子有冲击力
- [ ] 对白节奏自然
- [ ] 情绪弧线完整（结构模板落地）
- [ ] 结尾悬念有效
- [ ] 场景目的达成（推进剧情+揭示角色）
- [ ] 至少一次两难选择清晰可见

### 迭代精修（如果需要）

> 70% 合格、25% 需微调、5% 需重来。可定位问题优先精修而非整段重做。
> ⚠️ **视频重生成门禁**：涉及重新提交 Seedance 生成视频的精修方案，必须先向用户报告问题并获得明确同意。未经同意禁止重新生成视频。故事板帧（Seedream 图片）重新生成后也需要提交用户确认。

| 问题类型 | 精修方案 | 需要用户同意？ |
|----------|----------|--------------|
| 某帧角色表情不对 | 用 Seedream 图生图重生成该帧 | ❌ 不需要（图片） |
| 字幕错别字 | FFmpeg drawtext 局部覆盖 | ❌ 不需要（后期处理） |
| 局部色彩偏差 | FFmpeg colorbalance 校正 | ❌ 不需要（后期处理） |
| 某段视频画面质量不合格 | 调整 prompt 重新提交 Seedance | ✅ **必须用户同意** |
| 某段节奏偏慢/快 | 调整 `--duration` 重新生成 | ✅ **必须用户同意** |
| 分离光不足 | 重出该段 | ✅ **必须用户同意** |

```bash
# 精修示例：图生图重出单帧，保持构图
python3 <video-gen>/scripts/seedream.py create \
  -p "保持原构图和光影，仅修改角色表情为{目标表情}" \
  -i ./03-storyboard/ep01/seg1/frame03.png \
  --size 9:16 \
  -d ./03-storyboard/ep01/seg1/
```

### 提交终审

AI 自检完成后，向用户发送成片，附带：
- 自检结果摘要（通过/发现问题/已修复的问题）
- 需要用户关注的项目（仅音频相关）
- 如有未能自主修复的问题，列出具体位置和原因

> 用户只需看一遍成片，重点听音频。画面和字幕问题 AI 已自检处理。
