# 阶段 13：批量生产（后续集）

> 工具：循环阶段 05-11
> ⚠️ 逐集独立——新集重新生成故事板和视频，不沿用上一集素材（角色设定图、character-bible 和 Moodboard 除外）

## 前置条件

- 第 1 集成片已通过检查
- 角色设定图、Moodboard、灯光哲学、配色方案作为常量沿用

## 流程

### 对每一集循环执行

1. **阶段 05** — 剧本（新集）
2. **阶段 06** — 故事板（新集）
3. **阶段 07** — 四段视频（prompt 组装 + 串行提交）
4. **阶段 11** — 合成
5. **阶段 12** — 成片检查

### 每集独立原则

- 新集首帧重新 Seedream 生成，不沿用上集尾帧
- 不同时提交多个 Seedance 任务（串行提交）
- 每集完整体验确认后进入下一集

### 每 5 集：跨集一致性审计

暂停批量生产，执行以下审计。

#### 视觉审计

取第 1 集和第 N 集同角色面部特写帧，目测对比：

```bash
cp ./03-storyboard/ep01/seg1/*特写*.png ./07-consistency-check/faces/
cp ./03-storyboard/ep0N/seg1/*特写*.png ./07-consistency-check/faces/
```

- [ ] 眼型、发色、脸型是否一致
- [ ] 同类型场景色调是否漂移
- [ ] 配色方案语义标记是否持续使用

**修复漂移**：将第 1 集角色设定图重新挂入后续 `--ref-images` 优先级首位，prompt 中加 "严格按照角色设定图外观生成，不引入风格漂移"。

#### 声音审计（🎤 如有 voice-ref）

**Step 1 — 建立金标准（第 1 集）**

从第 1 集成片截取每个角色代表性对白片段：

```bash
ffmpeg -i ./06-output/ep01-final.mp4 \
  -ss {对白起始秒} -t {对白时长} \
  -vn -acodec copy \
  ./07-consistency-check/audio/{角色名}_ep01.mp3
```

**Step 2 — 对比审计（每 5 集）**

从第 N 集成片截取同角色对白：

```bash
ffmpeg -i ./06-output/ep0N-final.mp4 \
  -ss {对白起始秒} -t {对白时长} \
  -vn -acodec copy \
  ./07-consistency-check/audio/{角色名}_ep0N.mp3
```

人工对比：

| 检查项 | 判断标准 |
|---|---|
| 音色一致性 | 闭眼听，同角色像同一个人吗？ |
| 角色区分度 | A 和 B 还明显不同吗？ |
| 情感表现力 | 同样标记 angry/悲伤，还一样有张力吗？ |
| 音量/音质 | 有没有变轻、变闷、变尖、变糊？ |

> 取至少 3 句不同情绪对比。

**修复漂移**：

| 漂移程度 | 方案 |
|---|---|
| 轻微 | 重新将 ep01 的 voice-ref.mp3 挂入后续 `--audio` |
| voice-ref 本身质量不够 | TTS 精调参数重新生成 voice-ref |
| 严重 | 切到 B 方案：TTS 逐句合成全部对白，FFmpeg 替换音频轨 |

审计通过后继续后续 5 集。
