# 阶段 12：成片检查

> 工具：人工审核（逐项核对）

## 前置条件

- `06-output/ep01-final.mp4` 存在
- 所有故事板关键帧、角色设定图可供对比

## 流程

### 逐项检查

#### 画面
- [ ] 角色外观与设定图一致
- [ ] 画风与 Moodboard 一致
- [ ] 景别切换节奏舒适
- [ ] 无画面撕裂/闪烁/变形
- [ ] 色彩/光影与灯光哲学一致

#### 光影专项
- [ ] 主光方向在镜头间保持一致（非叙事动机变化时）
- [ ] 光比与场景情绪匹配，无随机漂移
- [ ] 色温在整段内无意外跳变
- [ ] 角色面部曝光一致（不突然过亮或过暗）
- [ ] 分离光有效（角色不从暗背景中消失）
- [ ] 灯光弧线与剧本灯光节拍对齐
- [ ] 特写中眼神光存在且方向一致
- [ ] 实用光源有画面内动机（如灯光哲学选择现实主义）
- [ ] 灯光母题在关键帧中可见

#### 声音
- [ ] 配音清晰可辨，无含混吞字
- [ ] 角色声音区分度明显（A 和 B 闭眼能分辨）
- [ ] 🎤 各角色音色与 voice-ref.mp3 一致
- [ ] 情感表现符合剧本标注（愤怒够怒、悲伤不假）
- [ ] 语速节奏自然，与画面情绪同步
- [ ] 环境音效符合场景
- [ ] 音量无突变/爆音
- [ ] 对白无缺失（每句剧本对白都在成片中出现）

#### AI 生成专项
- [ ] 无手指/肢体变形（多指、扭曲、消失）
- [ ] 面部身份一致（同段内角色面部无漂移变脸）
- [ ] 无物理逻辑错误（衣服穿透、物品凭空消失、比例突变）
- [ ] 口型与配音同步程度可接受
- [ ] 无字幕/文字乱码（AI 生成的文字经常是无意义符号）
- [ ] 无非自然闪烁/帧间色彩跳变

#### 字幕
- [ ] 字幕出现/消失与配音同步
- [ ] 字幕内容与对白一致
- [ ] 无错别字
- [ ] 字幕不遮挡关键画面

#### 叙事
- [ ] 3 秒钩子有冲击力
- [ ] 对白节奏自然
- [ ] 情绪弧线完整（结构模板落地）
- [ ] 结尾悬念有效
- [ ] 场景目的达成（推进剧情+揭示角色）
- [ ] 至少一次两难选择清晰可见

### 迭代精修（如果需要）

> 70% 合格、25% 需微调、5% 需重来。可定位问题优先精修而非整段重做。

| 问题类型 | 精修方案 |
|----------|----------|
| 某帧角色表情不对 | 用 Seedream 图生图重生成该帧，挂原帧为 `-i`，仅改表情描述 |
| 字幕/文字乱码 | 如支持视频编辑：修改字幕区域文字 |
| 局部色彩偏差 | 调整 prompt 重出该段 |
| 某段节奏偏慢/快 | 调整 `--duration` 重新生成该段 |
| 分离光不足 | 重出该段，prompt 强化轮廓光/色彩分离描述 |

```bash
# 精修示例：图生图重出单帧，保持构图
python3 <video-gen>/scripts/seedream.py create \
  -p "保持原构图和光影，仅修改角色表情为{目标表情}" \
  -i ./03-storyboard/ep01/seg1/frame03.png \
  --size 9:16 \
  -d ./03-storyboard/ep01/seg1/
```
