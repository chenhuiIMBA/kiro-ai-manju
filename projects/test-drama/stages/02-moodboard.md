# 阶段 02：风格板（Moodboard）

> 工具：Seedream 5.0（图片生成）
> 前置：`ARK_API_KEY` 已设置，`video-generation` skill 可用

## 前置条件

- `01-scripts/series-bible.md` 已确认
- `03-storyboard/visual-style-spec.md` 存在

## 流程

### 步骤 1：生成风格概念图

> ⚠️ **Prompt 审核门禁**：组装 prompt 后，对照 `references/prompt-guide.md` §八「图片 Prompt 审查清单」逐项自查。然后将完整 prompt + 参数发送给用户审核，确认后才可提交。

用 Seedream 快速出 5-8 张不同场景的概念图。每个场景单独调用：

```bash
python3 <video-gen>/scripts/seedream.py create \
  -p "{画风}风格概念插画。{场景描述}，{光影描述}，{色调}。高品质概念插画。" \
  --size 9:16 \
  -d ./02-moodboard/
```

### 步骤 2：覆盖场景类型

根据题材选取 5-8 种，不强制全覆：

| 类型 | 示例 prompt | 灯光要点 |
|------|------------|---------|
| 白天外景 | `午后阳光透过树叶洒落地面，暖黄色调，温暖日常氛围` | 顶光/侧光，光比 2:1~3:1，暖色温 |
| 夜间内景 | `昏暗室内，窗外月光照入，冷蓝色调，神秘安静氛围` | 月光主光源（冷），可能有暖色人工光点缀 |
| 逆光/剪影 | `角色背对落日，身体剪影被金色光线勾勒轮廓，天空橙红渐变` | 主体曝光不足，背景高亮，高反差 |
| 人物特写 | `女性面部特写，侧光打在脸上，眼神深邃，暖橙与深蓝对比` | 柔光为主，眼神光必须存在 |
| 室内混合光源 | `咖啡厅靠窗，窗外阴天漫射冷光，桌面暖黄台灯补光` | 冷暖色温共存 |
| 大场景氛围 | `俯瞰城市夜景，万家灯火，深紫与金色交织` | 无单一主光，靠整体层次 |
| 黄金时刻 | `日落前30分钟，整个场景被金色暖光包裹，影子被拉长` | 低角度暖光，长阴影，色温 ~3500K |
| 蓝调时刻 | `日落后20分钟，天空深邃蓝色，地面灯光点点，冷暖强烈对比` | 天空冷蓝为主，地面暖光点缀 |
| 雨景/水景 | `雨中街道，地面湿漉反射霓虹灯光，雨滴在灯光中闪烁` | 多次反射/散射光源，高光点密集 |

### 步骤 2.5：避免千篇一律——风格变体轮换

> 同一个"日系动漫风格概念插画"跑 5 次会产出高度相似的光影和构图。每集/每项目的 Moodboard 必须从以下池子中**主动注入变体描述**。

#### 色彩策略轮换

不要每张图都用"暖黄色调"。从以下轮换，至少取 3 种：

| 色彩策略 | 描述短语 | 情绪 |
|---------|---------|------|
| 暖金主调 | "暖金色阳光包裹" | 温暖、希望、日常 |
| 冷蓝主调 | "冷蓝色调笼罩" | 孤独、理性、深夜 |
| 冷暖对冲 | "暖橙肤色与冷蓝背景形成强烈对比" | 内心挣扎、对峙 |
| 低饱和沉郁 | "去饱和的灰调，色彩被抽离" | 失落、回忆、压抑 |
| 高饱和鲜艳 | "色彩鲜明饱和，如插画般绚丽" | 幻想、青春、活力 |
| 单色渐变 | "单一色相渐变，从深蓝到浅蓝的层次" | 高度风格化、梦境 |

#### 构图表征轮换

不要每张概念图都用"角色站在画面中央"。至少混入 3 种：

| 构图策略 | 描述短语 | 适合 |
|---------|---------|------|
| 对称中央 | "角色位于画面正中央，对称构图" | 庄严、压迫感 |
| 三分之一偏移 | "角色偏移至右侧1/3，左侧留白" | 孤寂、思考、看向远方 |
| 极低角度 | "极低角度仰拍，角色占据画面上方2/3" | 强大、威胁、仰视感 |
| 俯瞰扁平 | "俯瞰视角，角色只是场景中的一个小点" | 渺小、宿命、全景叙事 |
| 前景遮挡 | "透过前景物体（门框/树叶/窗格）偷窥式构图" | 窥视、秘密、疏离 |
| 极端特写 | "只取眼睛或嘴唇的特写，背景全虚" | 情感爆发、内心戏 |

#### 光影氛围轮换

不要每张都用"阳光透过树叶"。从以下轮换：

| 氛围 | 描述短语 | 适用题材 |
|------|---------|---------|
| 晨雾 | "薄雾弥漫，光线柔和散射，空气感强" | 清新、悬疑 |
| 正午烈日 | "强烈顶光，阴影锐利短小，高反差" | 紧张、对峙 |
| 烛光/火光 | "单一烛光从下方照亮面部，周围全暗" | 秘密、亲密、恐怖 |
| 霓虹夜色 | "霓虹灯光在地面水渍中反射，多色污染" | 都市、赛博 |
| 阴天漫射 | "阴天均匀漫射光，无明确阴影，灰冷色调" | 压抑、日常、厌倦 |
| 闪电瞬间 | "闪电瞬间的白光照亮整个场景，影子极长" | 惊悚、转折 |

#### 使用规则

- 每集 Moodboard 启动时，从以上三个池子各取 1-2 种**未在之前用过**的组合
- 在 `02-moodboard/assets.md` 记录本轮使用的变体标签，供后续集参考避免重复

### 步骤 3：用户确认

提交概念图给用户确认——"就这个感觉"。

不满意则调整方向重新出。**确认后再进灯光哲学。**
