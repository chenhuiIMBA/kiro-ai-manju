# 第 6 集：彩排

**时长**：60 秒
**结构模板**：A（微三幕）
**钩子类型**：#2 反差揭示（苏晚宁以策划师身份检查婚礼——她策划过上百场婚礼，但这次她是新娘）
**冲突引擎**：身份错位（策划师 vs 新娘）+ 仪式感入侵（假戏中的真仪式让身体先于理性做出反应）
**对白节奏**：职业干练型（seg1 苏晚宁检查细节）→ 沉默-凝视型（seg2 陆深看到婚纱）→ 紧张-停顿型（seg3 交换戒指）→ 脱口而出型（seg4 苏晚宁说"你今天很帅"）
**集主题推进**：仪式感是催化剂——当你穿上婚纱、交换戒指、说出誓词，即使是假的，身体和情感也会产生真实反应。这集回答"假戏做到什么程度会开始变真"

---

## 🪝 钩子（0-3 秒）

**画面**：婚礼场地，白天。镜头不拍人，拍一双女性的手在整理婚礼签到台上的花艺——手法专业、快速、精准。然后这双手停下来，镜头缓缓拉远，露出苏晚宁穿着日常通勤装站在签到台前——她不是在工作，她是在检查自己的婚礼。
**对白/声音**：花瓣被拨动的细微声响，远处工人搬运桌椅的声音。
**设计意图**：用"策划师检查婚礼"的专业动作开场，然后揭示——这是她自己的婚礼。身份错位的冲击力：她策划过上百场别人的幸福，这次轮到她自己了，但这场是假的。

---

### 段 1（0-15 秒）— 钩子 + 布设

**场景目的**：①建立婚礼场地空间，展示苏晚宁的策划师本能 ②通过她的专业挑剔暗示她对"完美婚礼"有执念——这个执念和她的 Flaw 有关（她帮别人策划幸福，但不相信自己能拥有）③陆深到场，两人在婚礼场地的第一次对视

**情绪节拍**：专业检查(0-3s) → 发现问题，皱眉(3-6s) → 调整花艺(6-9s) → 陆深到场(9-12s) → 两人在花拱门下对视(12-15s)

**灯光节拍**：婚礼场地室内，大面积落地窗引入自然日光，暖白色调（wedding_day）。花拱门上缠绕的白色灯串尚未点亮但可见。整体明亮通透，光比约2:1，柔光。苏晚宁站在窗光中，光线从右侧打入，照亮她的侧脸和手上的动作

**冲突引擎**：身份错位（策划师在检查自己的假婚礼）

**场景描述**：高端婚礼场地，白天。挑高空间，左右两侧大面积落地窗，自然光充沛。中央过道尽头是一座白色花拱门，两侧摆放着尚未完成布置的宾客座椅。签到台在入口处，上面摆着精致的花艺。工人在远处搬运桌椅。

**镜头序列**：手整理花艺的特写（钩子）→ 镜头拉远露出苏晚宁全身中景 → 陆深从入口走入的远景 → 两人在花拱门前的中景对视

[开场3秒：一双女性的手在签到台上调整花艺——手指灵巧地拨开一朵偏了的白玫瑰，把它转到正确的角度。手法专业、快速。]

[镜头缓缓拉远，露出苏晚宁穿着白衬衫和卡其色风衣，站在签到台前。她不是在工作——她在检查自己的婚礼。]

苏晚宁（皱眉，自言自语，语气是职业习惯）："花拱门的灯串间距不对……左边密右边疏。"
> 潜台词：策划师的本能——即使是假婚礼也不能有瑕疵
> 对白行动：自我暴露（她对"完美婚礼"的执念）
> 🎤 emotion: neutral, speed: +5%, loudness: -10%

[她走向花拱门，伸手调整灯串。身后传来脚步声。]

陆深从入口走入，穿深灰西装，步伐稳定。他在过道中央停下，看到苏晚宁站在花拱门下——逆光，她的轮廓被窗光勾勒。

陆深（停下脚步，语气平淡）："你比婚庆公司来得还早。"
> 潜台词：你很在意这场婚礼（他注意到了）
> 对白行动：观察/试探
> 🎤 emotion: neutral, speed: -5%, loudness: 0

苏晚宁（转身，笑了一下，语气轻快）："职业病。看到不完美的婚礼会手痒。"
> 潜台词：我只是在工作（不是因为在意）
> 对白行动：掩饰
> 🎤 emotion: happy, speed: +10%, loudness: 0

### 段 2（15-30 秒）— 对抗·婚纱

**场景目的**：①苏晚宁换上婚纱出来的瞬间——这是全集的视觉高潮 ②陆深的反应是核心：他的表情变了，但很快恢复冷脸。这个"变了又恢复"比任何台词都有力 ③苏晚宁捕捉到了那个瞬间，但她不确定自己看到了什么

**情绪节拍**：更衣室门打开(15-17s) → 苏晚宁穿婚纱走出(17-20s) → 陆深看到她的瞬间——表情裂开(20-23s) → 他迅速恢复冷脸(23-25s) → 苏晚宁走近(25-28s) → 两人在镜子前并排站立(28-30s)

**灯光节拍**：更衣室方向偏暗，苏晚宁从暗处走向亮处——落地窗的自然光从右侧打入，婚纱的白色面料在光线中微微发光，半透明的裙摆边缘透出柔和的光晕。陆深站在光线中等她，当她走近时，两人同时被窗光照亮——这是全剧第一次两人在同一束光里

**冲突引擎**：仪式感入侵（婚纱不是普通衣服——它承载了"新娘"的全部象征意义。苏晚宁穿上它的瞬间，"假"和"真"的边界模糊了）

**场景描述**：同一婚礼场地。更衣室在场地侧面，门口有一面全身镜。陆深站在过道中央等待。

**镜头序列**：更衣室门打开的中景 → 苏晚宁穿婚纱走出的全身远景（逆光剪影到正面揭示）→ 陆深面部特写（表情变化——这是关键镜头）→ 两人在全身镜前的中景

[更衣室的门打开。]

[苏晚宁穿着白色婚纱走出来——不是华丽的公主裙，是简约的缎面A字裙，贴合身形，裙摆拖地。她的头发被简单地挽起，露出脖颈线条。她一只手提着裙摆，另一只手扶着门框，表情有点不自在——她穿过无数次婚纱给客户看效果，但这次不一样。]

[镜头切到陆深。他站在过道中央，双手插在裤袋里。看到苏晚宁的瞬间——他的表情变了。不是惊艳的那种变，是某种更深的东西：眼神从空白变成柔软，嘴唇微微张开，右手从裤袋里抽出来，手指微微蜷缩——像是想伸出去又克制住了。]

[这个表情只持续了两秒。然后他的下巴微微抬起，眼神恢复冷淡。]

苏晚宁走近，在他面前站定。她注意到了什么，但不确定。

苏晚宁（微微歪头，语气带一点试探）："怎么样？能过关吗？"
> 潜台词：你刚才的表情是什么意思
> 对白行动：试探（用轻松的问题探测他的反应）
> 🎤 emotion: neutral, speed: 0, loudness: 0

陆深（停顿一秒，目光从她脸上移开，看向别处）："……合适。"
> 潜台词：不只是合适（但他不会说）
> 对白行动：克制/回避
> 🎤 emotion: neutral, speed: -15%, loudness: -10%

[两人走到全身镜前并排站立。镜子里映出一对新人——西装笔挺的男人和白纱曳地的女人。苏晚宁看着镜中的两人，表情从不自在变成某种说不清的恍惚。]

### 段 3（30-45 秒）— 高潮·交换戒指

**场景目的**：①彩排走到交换戒指环节——仪式感最强的瞬间 ②两人的手指碰到一起时都顿了一下——身体的反应先于理性 ③这个"顿"是全集的情感高潮：不是心动的甜蜜，是"我不该有这种感觉"的恐慌

**情绪节拍**：司仪引导流程(30-33s) → 陆深拿起戒指(33-36s) → 他握住苏晚宁的手(36-39s) → 戒指滑上无名指——两人都顿了一下(39-42s) → 苏晚宁低头看戒指(42-44s) → 司仪说"很好"打破凝固(44-45s)

**灯光节拍**：花拱门下，两侧落地窗的自然光从左右同时打入，形成均匀的柔光包围。白色花拱门上的灯串被点亮（彩排测试），暖白色小灯泡在花朵间闪烁。两人站在光的中心——这是全剧灯光最"完美"的一刻，完美到像真的

**冲突引擎**：仪式感入侵的顶点（交换戒指=承诺的象征。即使是假的，手指碰到一起的触感是真的）

**场景描述**：花拱门下，灯串已点亮。司仪站在一旁拿着流程单。两人面对面站立。

**镜头序列**：花拱门下两人面对面的中景 → 陆深拿起戒指的手部特写 → 他握住苏晚宁手的中近景（两人的表情）→ 戒指滑上无名指的极近特写 → 苏晚宁低头看戒指的面部特写

[花拱门下，灯串亮着，暖白色光点在白色花朵间闪烁。]

司仪（站在一旁，看着流程单，语气职业化）："好，接下来是交换戒指环节。新郎先请。"
> 🎤 emotion: neutral, speed: 0, loudness: +5%

[陆深从司仪手中接过戒指盒，打开。里面是一枚简约的铂金戒指。他拿起戒指，转向苏晚宁。]

[他伸出左手，握住苏晚宁的右手。他的手指碰到她手指的瞬间——两个人都顿了一下。不是犹豫，是某种电流般的反应。]

[陆深的拇指无意识地在她的指节上停了一秒，然后他把戒指缓缓推上她的无名指。]

[苏晚宁低头看着戒指滑上自己的手指。她的嘴唇微微抿紧，睫毛低垂，表情不是喜悦——是某种她自己都说不清的复杂。]

[停顿 2 秒——两人都没有松手。花拱门上的灯串在他们头顶闪烁。]

司仪（打破沉默，语气轻松）："很好，很自然。明天就这样。"
> 🎤 emotion: happy, speed: +5%, loudness: 0

[苏晚宁先抽回了手。]

### 段 4（45-60 秒）— 揭示/悬念

**场景目的**：①彩排结束，两人换回便装，紧张感应该消退——但没有 ②苏晚宁脱口而出"你今天很帅"——这不是策划师对客户说的话 ③陆深愣了三秒没接话——他不是不会说话，是不知道怎么回应真心话

**情绪节拍**：换回便装走出场地(45-48s) → 苏晚宁随口说"你今天很帅"(48-51s) → 陆深愣住(51-54s) → 三秒沉默(54-57s) → 苏晚宁先走(57-59s) → 陆深看着她的背影(59-60s)

**灯光节拍**：场地外，傍晚。夕阳从左侧低角度打入，暖金色（golden_hour）。两人从室内走到室外，光线从人工暖白变成自然暖金——从"假"的婚礼灯光过渡到"真"的自然光。苏晚宁说"你今天很帅"时，夕阳正好照在她脸上，表情毫无防备

**冲突引擎**：脱口而出的真心话（苏晚宁的 Flaw 是"不依赖别人"，但她刚才说了一句不在"交易"范围内的话）+ 陆深的 Flaw 被触碰（有人在夸他本人，不是夸"陆家继承人"）

**场景描述**：婚礼场地外，傍晚。停车场边的人行道，两侧有行道树。夕阳低垂，暖金色光线穿过树叶间隙洒在地面上。

**镜头序列**：两人并肩走出场地的中景 → 苏晚宁说话时的中近景（表情自然，没有防备）→ 陆深愣住的面部特写（三秒）→ 苏晚宁先走的背影中景 → 陆深看着她背影的侧面中近景

[两人换回便装，并肩走出婚礼场地。苏晚宁穿回白衬衫和卡其色风衣，陆深穿深灰西装。傍晚的夕阳从左侧打来，暖金色光线穿过行道树洒在两人身上。]

苏晚宁（走着，语气随意，像在说今天天气不错）："你今天很帅。"
> 潜台词：不是客套，是真心话（但她自己可能还没意识到）
> 对白行动：脱口而出（不在计划内的真话）
> 🎤 emotion: happy, speed: +5%, loudness: 0

[陆深的脚步顿了一下。他转头看苏晚宁——她没有看他，目光看着前方，表情自然得像刚才那句话不需要任何回应。]

[停顿 3 秒——陆深没有说话。他的嘴唇动了一下，像是想说什么，但最终只是把目光移回前方。右手拇指在身侧无意识地搓了一下食指。]

💭 画面停在陆深侧脸的中近景上，夕阳暖金色光照在他脸上，表情不是冷漠——是某种不知所措。内心独白作为画外音出现——声音很低，像自言自语，带着困惑：
"她说的是……我？"

[苏晚宁先走了几步，回头看他还站在原地，笑了一下。]

苏晚宁（回头，语气轻松）："走啊，发什么呆。"
> 潜台词：你的反应比我预想的大（但她不会说）
> 对白行动：化解尴尬
> 🎤 emotion: happy, speed: +10%, loudness: +5%

[最后一个画面：陆深跟上去，两人并肩走在夕阳下的人行道上。他们的影子在地面上被拉得很长，影子的手几乎碰在一起——但没有。]

— 悬念：他是不习惯被夸，还是因为是她说的？而苏晚宁——她说"你今天很帅"的时候，是策划师在评价客户，还是一个女人在看一个男人？

## 段拆分

### 段 1（0-15 秒）
- 时间段：0s ~ 15s
- 结构定位：钩子 + 布设
- 出场角色：苏晚宁、陆深
- 场景：婚礼场地（白天）
- 对白台词：
  - 苏晚宁："花拱门的灯串间距不对……左边密右边疏。"
  - 陆深："你比婚庆公司来得还早。"
  - 苏晚宁："职业病。看到不完美的婚礼会手痒。"

### 段 2（15-30 秒）
- 时间段：15s ~ 30s
- 结构定位：对抗·婚纱
- 出场角色：苏晚宁、陆深
- 场景：婚礼场地·更衣室出口（白天）
- 对白台词：
  - 苏晚宁："怎么样？能过关吗？"
  - 陆深："……合适。"

### 段 3（30-45 秒）
- 时间段：30s ~ 45s

- 出场角色：苏晚宁、陆深、司仪（配角）
- 场景：花拱门下（白天，灯串点亮）
- 对白台词：
  - 司仪："好，接下来是交换戒指环节。新郎先请。"
  - 司仪："很好，很自然。明天就这样。"

### 段 4（45-60 秒）
- 时间段：45s ~ 60s

- 出场角色：苏晚宁、陆深
- 场景：婚礼场地外·傍晚
- 对白台词：
  - 苏晚宁："你今天很帅。"
