# 七日恋人 整季大纲

## 主题

"真正的亲密不是完美的表演，而是敢于在对方面前暴露自己的不完美。"

两个各怀目的的人被迫扮演恩爱夫妻，在表演亲密的过程中意外触碰到真实的亲密——而真实的亲密比表演更可怕，因为它意味着脆弱。

## 中心戏剧问题

苏晚宁和陆深能在"假婚礼"结束后，承认自己在表演中动了真心吗？

## 系列大弧

从"两个人各怀目的地演戏" → 到"演着演着分不清真假" → 到"真相暴露后的信任崩塌" → 到"放下防备，选择真实"。

## 人物弧

### 苏晚宁（女主）
- **Want**：拿到婚礼策划尾款，还清母亲的手术费
- **Need**：学会接受帮助，不再一个人扛所有事
- **Flaw**：坚信"依赖别人就会被抛弃"——所以所有事都自己扛，拒绝示弱
- **Ghost**：父亲在她 10 岁时抛弃家庭，从此她成为家里的顶梁柱
- 起始状态：独立、倔强、嘴硬心软，习惯性拒绝好意
- 成长路径：被迫接受陆深的"假关心" → 发现假关心里有真心 → 害怕再次被抛弃而推开他 → 最终选择信任
- 终局状态：学会说"我需要你"

### 陆深（男主）
- **Want**：用一场完美婚礼堵住家族逼婚的压力，维持自由
- **Need**：承认自己渴望被人真正在乎，而不是被当作"陆家继承人"
- **Flaw**：坚信"所有人靠近我都是因为我的身份"——所以用冷漠和控制保持距离
- **Ghost**：母亲去世后，父亲迅速再婚，继母对他好只是为了家产。从此他不相信任何人的"好意"
- 起始状态：冷面、控制欲强、习惯用钱和权力解决问题
- 成长路径：发现苏晚宁是唯一不在乎他身份的人 → 开始主动靠近 → 被苏晚宁推开后第一次感到无力 → 放下控制，用真心换真心
- 终局状态：学会说"留下来，不是因为交易"

## 角色关系图

```
苏晚宁 ←——契约假夫妻——→ 陆深
  ↑ 闺蜜/同事              ↑ 发小/商业伙伴
林可欣                    周彦（暗线反派）
  （支持者）               （搅局者，陆深前任的哥哥）
```

- 苏晚宁 ↔ 陆深：契约关系 → 假戏真做 → 信任危机 → 真心相许
- 苏晚宁 ↔ 林可欣：闺蜜，苏晚宁的情感出口，偶尔推波助澜
- 陆深 ↔ 周彦：表面商业伙伴，实际是前任的哥哥，暗中想拆散陆深的"婚姻"让妹妹回来

## 冲突主线

- A 故事线（主线）：假婚礼契约 → 假戏真做 → 信任危机 → 真心相许
- B 故事线（支线）：周彦+许若晴的商业阴谋 → 利用感情裂缝夺取股权 → 被揭穿

## 戏剧反讽设计

观众从第 4 集就知道陆深在暗中帮苏晚宁（付医药费），但苏晚宁到第 14 集才知道。这 10 集的信息差让观众一直在喊"你快看看他对你多好啊"——这就是追更动力。
