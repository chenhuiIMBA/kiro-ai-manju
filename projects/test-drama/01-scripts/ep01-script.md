# 第 1 集：新娘跑了

**时长**：60 秒
**结构模板**：A（微三幕）
**钩子类型**：#2 对话开场
**冲突引擎**：两难选择（接受交易=出卖尊严，拒绝=母亲手术费没着落）
**对白节奏**：加速型（句越来越短，停顿越来越少，到 seg3 达到峰值）
**集主题推进**：建立核心交易——两个各怀目的的人因为一场意外被绑在一起，回答"他们是怎么开始的"

---

## 🪝 钩子（0-3 秒）

**画面**：手机屏幕特写，来电显示"陆氏婚礼-紧急"，屏幕光映在苏晚宁的脸上，她正在吃便利店饭团，嘴里还没咽下去就接了电话。
**对白/声音**：电话那头一个慌张的女声："苏老师！新娘跑了！"
**设计意图**：用最日常的画面（吃饭团）撞上最不日常的消息（新娘跑了），反差制造钩子。观众在 3 秒内获得核心信息：她是婚礼策划师，出事了。

---

### 段 1（0-15 秒）— 微三幕·钩子+布设

**场景目的**：①建立苏晚宁的职业身份和经济困境 ②用"新娘跑了"这个意外事件启动全剧

**情绪节拍**：日常松弛(0-3s) → 惊愕紧张(3-5s) → 快速切换到职业模式(5-15s)

**灯光节拍**：便利店冷白荧光灯（hospital_cold 近似）→ 出门后街道暖金路灯（warm_key），从冷到暖的快速切换暗示她从"自己的世界"冲向"别人的世界"

**冲突引擎**：环境压迫（时间紧迫，婚礼明天就办）

**场景描述**：便利店门口，深夜，苏晚宁穿着卡其色风衣（charA_primary），手里攥着吃了一半的饭团。街道上路灯昏黄，她边跑边打电话。

**镜头序列**：手机屏幕特写 → 苏晚宁脸部中景（嘴里有饭团的狼狈感）→ 她冲出便利店的全身跟拍

苏晚宁（惊愕，嘴里还在嚼东西）："……跑了？什么叫跑了？"
> 潜台词：这不可能，我所有流程都确认过了
> 对白行动：确认信息
> 🎤 emotion: surprise, speed: +15%, loudness: +10%

电话女声（慌张）："留了一封信就走了！陆先生那边已经知道了，他让你现在过来！"

苏晚宁（迅速冷静，扔掉饭团，开始跑）："宾客通知了吗？场地那边——算了，我十五分钟到。"
> 潜台词：先解决问题，崩溃的事之后再说
> 对白行动：掌控局面
> 🎤 emotion: neutral, speed: +20%, loudness: 0

[苏晚宁拦下一辆出租车，车门关上的声音]

---

### 段 2（15-30 秒）— 微三幕·对抗前半

**场景目的**：①苏晚宁和陆深第一次见面，建立两人的气场对比 ②陆深抛出"交易"，观众第一次看到他的控制欲

**情绪节拍**：紧张赶到(15-17s) → 被陆深的冷漠震住(17-20s) → 从震惊中回过神(20-25s) → 愤怒拒绝(25-30s)

**灯光节拍**：豪宅客厅暖白色调（mansion_day 偏夜景 mansion_night），落地窗外城市夜景冷蓝（cool_key），陆深站在窗前背光——他的脸半明半暗，苏晚宁站在门口被暖色吊灯照亮。光线从陆深的左侧打入（左侧=过去，暗示他此刻被家族的过去困住）

**冲突引擎**：信息不对等（陆深已经想好了方案，苏晚宁完全不知道）

**场景描述**：陆深家客厅，空间开阔，深色家具，落地窗外是城市夜景。陆深穿深藏蓝西装（charB_primary），站在窗前，背对门口。苏晚宁推门进来，风衣上还带着夜风的褶皱。

**镜头序列**：苏晚宁推门进入的中景 → 陆深背影的远景（建立空间压迫感）→ 陆深转身的中近景（第一次看到他的脸）→ 苏晚宁的反应特写

苏晚宁（喘着气，职业模式）："陆先生，我已经在路上联系了备选司仪和——"
> 潜台词：我能搞定，这是我的工作
> 对白行动：展示专业能力
> 🎤 emotion: neutral, speed: +10%, loudness: 0

陆深（转身，表情平静得不正常，声音很低）："不用了。"
> 潜台词：你说的那些都不重要
> 对白行动：掌控/截断
> 🎤 emotion: coldness, speed: -10%, loudness: -10%

[停顿 1.5 秒 — 苏晚宁被他的平静震住，她预期的是慌张，不是这个]

陆深（目光直视苏晚宁，右手拇指无意识地转着左手无名指的银戒）："婚礼照常办。我需要一个新娘。"

苏晚宁（愣住，然后皱眉）："……您的意思是临时找人？婚礼明天就——"
> 潜台词：这人疯了吗
> 对白行动：质疑
> 🎤 emotion: surprise, speed: 0, loudness: 0

陆深："不是找人。是你。"
> 潜台词：我已经想好了，你没有讨价还价的余地
> 对白行动：命令
> 🎤 emotion: coldness, speed: -15%, loudness: 0

---

### 段 3（30-45 秒）— 微三幕·对抗后半+高潮

**场景目的**：①苏晚宁拒绝交易，展示她的底线和倔强 ②陆深用数字击中她的软肋，展示他"用钱解决问题"的 Flaw

**情绪节拍**：愤怒拒绝(30-33s) → 陆深步步紧逼(33-38s) → 数字砸下来的沉默(38-40s) → 内心挣扎(40-45s)

**灯光节拍**：苏晚宁后退时从暖光区退向门口的暗区，陆深走近时从背光走向正面光——他从暗处走出来，像是在用光线逼近她

**冲突引擎**：两难选择（接受=出卖尊严，拒绝=母亲手术费没着落）

**场景描述**：同一客厅，两人距离在缩短。苏晚宁本能后退半步（标志性动作），陆深缓步走近。

**镜头序列**：两人对峙的中景 → 苏晚宁后退的脚步特写 → 陆深递出支票的手部特写 → 苏晚宁脸部特写（眼神从愤怒变成动摇）

苏晚宁（后退半步，侧头避开他的目光——标志性动作）："陆先生，我是婚礼策划师，不是——这不在服务范围内。"
> 潜台词：你把我当什么了
> 对白行动：划定边界
> 🎤 emotion: anger, speed: +10%, loudness: +5%

陆深（停下脚步，从西装内袋取出一张支票放在茶几上）："七天。婚礼结束，各走各路。策划尾款全额支付，另加补偿金。"
> 潜台词：这是交易，不是侮辱，我在给你台阶
> 对白行动：谈判/施压
> 🎤 emotion: coldness, speed: -10%, loudness: -5%

苏晚宁（没看支票）："你找错人了。"
> 潜台词：我不是可以用钱买的
> 对白行动：拒绝
> 🎤 emotion: anger, speed: 0, loudness: 0

陆深（声音没有起伏，像在念一个数字）："五十万。"
> 潜台词：我知道你需要这个数字
> 对白行动：精准打击
> 🎤 emotion: coldness, speed: -20%, loudness: -10%

[停顿 2 秒 — 苏晚宁的手指无意识地摸向手腕上的旧发绳（心虚/动摇时的标志性动作）。她的眼神从愤怒变成空白，再变成痛苦。五十万，刚好是母亲手术费的缺口。]

---

### 段 4（45-60 秒）— 微三幕·高潮+揭示/悬念

**场景目的**：①苏晚宁做出选择，签下契约 ②用"空房间"埋下第一个伏笔——这段婚约本来就是假的

**情绪节拍**：沉默中的妥协(45-48s) → 签字的决绝(48-52s) → 走进新娘房间的不安(52-57s) → 发现异常的困惑(57-60s)

**灯光节拍**：签字时茶几上的台灯暖光（mansion_night）照亮两人的手——暖光下签冷酷的契约，反讽。走进新娘房间后，灯光变为冷白月光（cool_key）从窗户照入，空荡荡的房间被冷光笼罩

**冲突引擎**：内在矛盾（苏晚宁想要钱但不想依赖任何人，签字本身就是对她 Flaw 的第一次冲击）

**场景描述**：客厅 → 走廊 → 新娘房间。苏晚宁签完字后被管家引到二楼的房间。房间很大，床铺整洁，但——没有任何私人物品。没有照片，没有化妆品，没有衣柜里的衣服。像一个从未被人住过的样板间。

**镜头序列**：苏晚宁低头看支票的俯拍 → 笔尖触纸的极近特写 → 苏晚宁走进房间的背影中景 → 空荡房间的慢摇镜头 → 苏晚宁站在房间中央的远景（人很小，房间很大很空）

苏晚宁（低头看着支票上的数字，声音很轻）："……七天？"
> 潜台词：我在确认这个牢笼的期限
> 对白行动：确认条件
> 🎤 emotion: sadness, speed: -15%, loudness: -20%

陆深："七天。"
> 潜台词：仅此而已（但他的手指还在转银戒——他也不确定）
> 对白行动：承诺/约束
> 🎤 emotion: coldness, speed: -10%, loudness: -15%

[苏晚宁拿起笔，签字。笔尖在纸上的声音。]

苏晚宁（签完，站起来，恢复了职业表情）："合同我带走一份。还有，明天的流程我需要重新过一遍。"
> 潜台词：这是工作，不是别的，我不会让自己觉得这是别的
> 对白行动：重建边界
> 🎤 emotion: neutral, speed: +5%, loudness: 0

[场景转换：苏晚宁跟着管家走进二楼房间。门打开。]

[3 秒慢摇镜头：空荡荡的房间，整洁的床，空的衣柜，空的梳妆台。窗外月光冷白（cool_key）。]

苏晚宁（站在房间中央，环顾四周，眉头微蹙，自言自语）："……连个行李箱都没有？"
> 潜台词：这个"新娘"从来就没打算住进来过
> 对白行动：发现异常
> 🎤 emotion: surprise, speed: -10%, loudness: -25%

[最后一个画面：苏晚宁站在空房间中央，月光从窗户照进来，她的影子被拉得很长。她的手又摸上了手腕的发绳。]

— 悬念：这段婚约本来就是假的？那个跑掉的新娘，是不是从一开始就不打算嫁？

## 段拆分

### 段 1（0-15 秒）
- 时间段：0s ~ 15s
- 结构定位：微三幕·钩子 + 布设
- 出场角色：苏晚宁、电话女声（画外音）
- 场景：便利店门口 → 街道 → 出租车
- 对白台词：
  - 电话女声："苏老师！新娘跑了！"
  - 苏晚宁："……跑了？什么叫跑了？"
  - 电话女声："留了一封信就走了！陆先生那边已经知道了，他让你现在过来！"
  - 苏晚宁："宾客通知了吗？场地那边——算了，我十五分钟到。"

### 段 2（15-30 秒）
- 时间段：15s ~ 30s
- 结构定位：微三幕·对抗前半
- 出场角色：苏晚宁、陆深
- 场景：陆深家客厅
- 对白台词：
  - 苏晚宁："陆先生，我已经在路上联系了备选司仪和——"
  - 陆深："不用了。"
  - 陆深："婚礼照常办。我需要一个新娘。"
  - 苏晚宁："……您的意思是临时找人？婚礼明天就——"
  - 陆深："不是找人。是你。"

### 段 3（30-45 秒）
- 时间段：30s ~ 45s
- 结构定位：微三幕·对抗后半 + 高潮
- 出场角色：苏晚宁、陆深
- 场景：陆深家客厅（两人距离缩短）
- 对白台词：

  - 苏晚宁："你找错人了。"
  - 陆深："五十万。"

### 段 4（45-60 秒）

- 结构定位：微三幕·高潮 + 揭示/悬念
- 出场角色：苏晚宁、陆深、管家（无台词）
- 场景：客厅 → 二楼新娘房间
- 对白台词：
  - 苏晚宁："……七天？"
  - 陆深："七天。"
  - 苏晚宁："合同我带走一份。还有，明天的流程我需要重新过一遍。"
