# 第 7 集：第七天

**时长**：60 秒
**结构模板**：A（微三幕）
**钩子类型**：#3 细节特写（镜中倒影里一只拿口红的手在发抖——不揭示是谁，不揭示为什么）
**冲突引擎**：仪式催化（正式婚礼把 EP06 彩排中的身体反应升级为灵魂反应）+ 语言突破（陆深第一次用语言而非行动表达真心）+ 沉默对峙（契约到期但无人提起）
**对白节奏**：独白-颤抖型（seg1 苏晚宁镜前独处）→ 脱稿-震撼型（seg2 陆深誓词脱稿）→ 沉默-停顿型（seg3 交换戒指）→ 日常-伪装型（seg4 用"饿了吗"回避告别）
**集主题推进**：当你在假婚礼上说出真心话，你还能假装这是交易吗？EP06 是身体先于理性，EP07 是语言先于理性——陆深的 Flaw 在仪式催化下第一次以语言形式裂开。而苏晚宁的 Flaw 在最后一段以另一种方式裂开：她没有说"契约到期了"，而是说"饿了吗"——她在用日常感替代告别，因为她不想走

**伏笔兑现**：
- EP02-seg4「陆深牵手时手在发抖」→ seg3 苏晚宁注意到他的手不抖了（从紧张到确定）
- EP06-seg2「陆深看到婚纱时表情失控两秒」→ seg2 誓词脱稿（视觉失控升级为语言失控）
- EP06-seg3「交换戒指时两人都没有松手」→ seg3 陆深握住苏晚宁的手没有松开（从被动反应到主动握住）

**新埋伏笔**：
- seg2 陆深脱稿的那句话 → EP13（苏晚宁离开时回想）
- seg3 苏晚宁停了一秒 → EP13（留下戒指时的犹豫）
- seg4 两人都没提契约到期 → EP08（"多余的一天"的前提）
- seg4 两人都没摘戒指 → EP13（苏晚宁把戒指留在床头柜）

### 段 1（0-15 秒）— 钩子 + 布设：镜中的新娘

**情绪节拍**：手抖+镜中倒影(0-3s) → 深呼吸，试图稳住(3-6s) → 画完口红，看着镜中的自己(6-9s) → 内心独白(9-11s) → 林可欣推门进来(11-13s) → 苏晚宁收起表情(13-15s)

### 段 2（15-30 秒）— 对抗·誓词脱稿

**场景目的**：①婚礼仪式进行中，两人在宾客面前表演完美——但誓词环节是裂缝 ②苏晚宁先念台本誓词，标准得体，无懈可击——这是她的安全区（用专业掩盖真心） ③陆深念了两句台本后停下来，看着苏晚宁的眼睛说了一句脱稿的话——不是情话，是一句只有他们两个人能听懂的真话。这是他的 Flaw 第一次以语言形式裂开 ④苏晚宁的表情从"表演微笑"变成"真的愣住了"——她听懂了

**情绪节拍**：婚礼仪式进行中(15-17s) → 苏晚宁念台本誓词(17-21s) → 陆深开始念台本(21-23s) → 他停下来(23-24s) → 脱稿说出真话(24-27s) → 苏晚宁愣住(27-29s) → 宾客鼓掌(29-30s)

**灯光节拍**：花拱门下，白天。与 EP06 seg3 相同的场地但今天是正式婚礼——灯串全部点亮，花艺完整，宾客就座。大面积落地窗自然光从两侧打入，花拱门上的暖白灯串补光，整体明亮柔和。两人站在光的中心，被暖白光包围。陆深脱稿时，镜头推近他的面部——化妆灯和窗光在他眼中形成清晰的眼神光（catchlight）

**冲突引擎**：台本 vs 脱稿（苏晚宁用台本保护自己，陆深用脱稿撕开保护）。宾客听到的是得体的誓词，苏晚宁听到的是真心话——信息不对等

**场景描述**：婚礼场地，花拱门下。正式婚礼进行中。两侧宾客就座，花拱门上白色花朵和暖白灯串交织。司仪站在一旁。苏晚宁穿婚纱，陆深穿深灰三件套西装。两人面对面站在花拱门下。

**情绪节拍**：司仪说"亲吻新娘"(30-32s) → 两人对视(32-34s) → 陆深没有吻她，而是伸手握住她的手(34-37s) → 苏晚宁感受到他的手——不抖了(37-39s) → 内心独白(39-40s) → 陆深侧头在她耳边说话(40-43s) → 苏晚宁眼眶泛红(43-44.5s) → 宾客鼓掌(44.5-45s)

**灯光节拍**：与 seg2 连续，同一场地同一光线。花拱门灯串的暖白光从上方洒落。陆深侧头靠近苏晚宁耳边时，他的脸进入她的阴影区——半明半暗，暗示这句话是私密的

**冲突引擎**：确定 vs 恐慌（陆深的手不抖了=他确定了；苏晚宁感受到这个变化=她的 Flaw 被触碰——有人在认真对待她）

**场景描述**：同一场地，花拱门下。紧接上一段。

**镜头序列**：两人面对面的中近景 → 陆深伸手握住苏晚宁的手的中景 →[推镜] 两人牵着的手的中近景 → 苏晚宁面部特写（感受到手不抖了）→ 陆深侧头靠近她耳边的侧面中近景

### 段 4（45-60 秒）— 揭示/悬念：沉默

**场景目的**：①婚礼结束，回到家。客厅里只有两个人——所有宾客、司仪、林可欣都走了，"表演"结束了 ②两人都换回便装，但都没有摘戒指——这个细节比任何台词都响 ③沉默。谁都没提"七天到了"、"契约到期了"。苏晚宁先开口，但说的不是告别——她说"饿了吗"，像在假装今天和其他任何一天没有区别 ④陆深的反应：不是回答"饿"或"不饿"，而是一个极微小的嘴角弧度——释然。他听懂了：她没有要走

**情绪节拍**：两人回到家(45-48s) → 客厅里站着，沉默(48-51s) → 苏晚宁看了一眼手上的戒指(51-53s) → 沉默持续(53-55s) → 苏晚宁说"饿了吗"(55-57s) → 陆深看着她(57-59s) → 最后画面：两只戴着戒指的手(59-60s)

**灯光节拍**：陆深家客厅，夜晚。室内暖黄色灯光（warm_key），光线柔和，光比约2:1。落地窗外是城市夜景的冷蓝色（night_ambient），冷暖对比——室外是冷的世界，室内是暖的。两人站在客厅中央，被暖光包围。最后一个画面：两只手垂在身侧的特写，暖黄灯光照在戒指的铂金表面上，反射出微小的光点

**冲突引擎**：沉默对峙（契约到期了，但没人提。苏晚宁的 Flaw 告诉她"该走了，不要依赖"，但她说了"饿了吗"——她在用日常感替代告别，因为她不想走。陆深的 Flaw 告诉他"她留下也是因为交易"，但他看到她没摘戒指——他选择不问，选择相信）

**场景描述**：陆深家客厅，夜晚。宽敞的现代风格客厅，落地窗外是城市夜景。室内暖黄色灯光。沙发、茶几、一切都整洁。两人从婚礼回来，已经换回便装——苏晚宁穿白衬衫，陆深穿深色休闲衫。

[客厅。夜晚。暖黄色灯光。]
