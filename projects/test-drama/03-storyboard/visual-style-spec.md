# 七日恋人 画风规格

## 画风定位

现代数字韩漫风，面向 18-25 岁女性受众。精致、成熟、理想化但不失真实感。

## 细化维度

| 维度 | 选定方向 |
|------|---------|
| 年代参照 | 当代（2020s 韩漫主流风格） |
| 风格参照 | 韩国 Webtoon 恋爱漫画风——如《女神降临》《某天成为公主》的视觉基调 |
| 线条特征 | 纤细均一，干净利落，无明显笔触变化 |
| 上色方式 | 软边渐变，柔和过渡，色彩饱和度中高 |
| 阴影风格 | 柔和阴影，2-3 层明暗分层，阴影边缘过渡自然 |
| 面部风格 | 精致写实但略带理想化——五官比例偏美型，眼睛大而有神但不夸张 |
| 身体比例 | 偏修长，头身比约 1:7.5~8，符合韩漫审美 |

## 对话呈现

字幕模式。对白以画面底部白色字幕呈现，关键情绪台词可加大字号。不使用气泡。

## Prompt 中的画风描述

后续所有 Seedream/Seedance prompt 统一使用以下画风前缀：

```
现代韩漫风格，精致写实画风，纤细线条，柔和渐变上色，面部精致美型，柔和阴影。
```

## 目标受众

18-25 岁女性，泛二次元 + 韩漫读者 + 短剧用户。
