# 本宫想开店 灯光哲学

> 本文档定义贯穿全剧 80 集的统一灯光 DNA。不是每场的具体灯光方案，而是所有场景必须遵守的光影原则。
> 后续阶段（04 场景库、05 角色设计、07 视频生成）的 prompt 必须引用本文档的术语和规则。

---

## 灯光基调

- **高调/低调**：**混合偏高调**——日常戏份（御膳房做菜、御花园散步、闺蜜互动）使用高调明亮画面；权谋戏份（太后佛堂、深夜密谈、朝堂对峙）使用低调暗画面。整体偏高调（甜宠+美食+喜剧的底色），低调仅在权谋/转折时出现
- **影调**：**柔阴影为主**——符合"三渲二"国风 3D 动漫的质感（非硬边赛璐珞，非写实硬阴影）。阴影边缘柔和渐变，2-3 层明暗过渡。仅在极端戏剧性场景（鸿门宴、太后杀意、萧珩两难选择）允许局部硬阴影

---

## 色彩策略

- **主导色温**：**暖色主导**——全剧 70% 画面偏暖（美食、日常、感情线），30% 偏冷（权谋、孤独、压迫）
- **冷暖配比规则**：
  - A 线（作死/日常）：暖金为主，偶尔冷暖对冲
  - B 线（感情线）：暖金渐进加深（从淡暖到浓暖，随感情升温）
  - C 线（权谋线）：冷蓝/深紫为主，暖光仅作为"人性残存"的点缀
  - D 线（身世线）：冷暖对冲最强烈——真相揭露时冷暖撕裂
- **色彩对比策略**：**冷暖共存**——同一画面内允许冷暖光源共存（如御膳房灶火暖橙 + 窗外月色冷蓝），这是本剧的核心视觉签名

## 灯光母题（贯穿全剧的灯光签名）

### 母题 1：灶火暖光 = 安全/归属

每当沈鹿溪在御膳房做菜时，画面中必须有灶火的暖橙色光源从下方/侧方照亮她的面部。这束暖光是她的"安全区"视觉符号——无论外面的权谋多冷，灶火永远是暖的。

- EP05 深夜做面：灶火暖光首次出现
- EP19 中秋宴做菜：灶火暖光最强烈
- EP48 给柳如烟煮阳春面：灶火暖光延伸到另一个人
- EP77 鸿门宴收官：灶火暖光最后一次在御膳房出现
- EP80 鹿溪记食肆：灶火暖光在新灶台上重生

### 母题 2：佛珠停止 = 太后做了决定

每当太后停止拨佛珠时，画面光线必须同步变化——从柔和漫射突然变为定向硬光（如一束顶光从上方打下），暗示"判决已下"。这是太后专属的灯光签名。

- 佛珠慢拨：柔和散射光，画面偏暖（伪装慈祥）
- 佛珠停止：光线骤变为冷色顶光，阴影加深（真面目）
- 佛珠断裂（EP76）：所有光源瞬间变为惨白——她的权力崩塌

## 实用光源哲学

**混合主义**：日常场景使用实用光源驱动（所有可见光必须有画面内光源动机——灶火、油灯、窗外阳光、宫灯），情绪转折允许表现主义（无动机光源，但必须与情绪一致）。

具体规则：
- 御膳房：灶火 + 油灯 + 窗外光（三光源标配）
- 太后佛堂：佛前烛光 + 窗外漫射（双光源）
- 冷月宫：月光 + 一盏孤灯（双光源，偏冷）
- 朝堂：高窗漫射 + 柱廊投影（大面积均匀光）
- 宫道/御花园：自然光为主（日光/月光）

---

## 曝光策略

- **曝光基准**：以场景中最重要的视觉元素为基准——人物戏以面部皮肤为基准，空镜以建筑/食材为基准
- **高光策略**：允许局部过曝——仅在以下场景：
  - 黄金时刻的逆光（mood_07 的"近暗远亮"效果）
  - 灶火火焰本身（火焰可过曝，但灶台细节保留）
  - 鸿门宴佛珠断裂瞬间的惨白闪光
- **暗部策略**：**保留暗部细节**（lifted）——本剧不是恐怖片，即使权谋戏也要让观众看清角色表情。暗部不纯黑，保留 10-15% 的细节可见度
- **曝光弧线**：
  - 第一幕（EP01-20）：整体偏亮（高调喜剧）
  - 第二幕（EP21-40）：逐渐降曝（卷入权谋）
  - 第三幕（EP41-60）：最暗（风暴中心）
  - 第四幕（EP61-80）：从暗回亮（她的选择），EP80 回到第一幕的亮度

---

## 分离光策略

- **轮廓光**：**不使用固定轮廓光**——本剧追求"三渲二"质感而非纯 2D 动画感，角色分离依赖色彩对比和景深虚化，而非硬轮廓光
- **背景光分离**：以**色彩对比**为主——
  - 暖色角色 + 冷色背景（最常用：沈鹿溪暖色衣裙 vs 冷色宫墙）
  - 冷色角色 + 暖色背景（太后深紫 vs 佛堂烛光暖色）
  - 景深虚化辅助（特写镜头背景虚化增强分离）

---

## 特殊灯光规则

1. **眼神光（必须）**：所有角色面部特写和中近景必须有眼神光——瞳孔中至少一个高光点，位置在 10 点钟或 2 点钟方向
2. **食物光（必须）**：所有食物特写必须有"食欲光"——从侧上方 45° 打入的暖光，让食材表面产生光泽和质感（参考 mood_05 的食材光影）
3. **阳春面专属光**：每次阳春面出现时，碗面必须有一圈暖金色的汤面反光——这是阳春面作为情感符号的视觉锚点
4. **过肩镜头规则**：前景肩膀可入暗（不超过 70% 暗），对面角色面部必须正确曝光

## Prompt 中的灯光描述模板

后续阶段写 prompt 时，灯光部分统一使用以下语义描述格式：

```
光影：{光源描述}，{色温方向}，{光比描述}，{阴影描述}，{特殊效果}
```

示例：
- "光影：灶火从左下方照亮面部，暖橙色温，强反差，柔和阴影边缘，热气中体积光散射"
- "光影：月光从窗外照入，冷蓝色温，中等反差，柔和散射阴影，薄雾弥漫增加空气感"
- "光影：正午顶光直射，暖金色温，极强反差，锐利短阴影，无特殊效果"
