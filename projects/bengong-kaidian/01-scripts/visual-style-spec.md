# 本宫想开店 视觉风格规格

## 核心定位

**国风 3D 动漫风（三渲二 × 架空古风）**

参照作品：《原神》过场 CG、《崩坏：星穹铁道》《鸣潮》《逆水寒》CG、《灵笼》系列。

## 五维规格

| 维度 | 选定方向 | 说明 |
|------|---------|------|
| 年代参照 | 2020s 游戏 CG / 国风 3D 动漫 | 非赛璐珞，非 2000s 早期 3D |
| 作品/工作室参照 | 米哈游 CG 风 + 网易《逆水寒》写实二次元基调 | 角色建模精致、场景恢宏、光影电影化 |
| 线条特征 | 软边轮廓（3D 模型自然边缘）+ 局部描边强化（眼睛、发丝、服饰花纹） | 非硬描边卡通渲染，非完全无线条 |
| 上色方式 | 3D 光照（PBR 材质）+ 二次元色块简化 | 皮肤/服饰有质感但避免毛孔级真实；阴影 2-3 层柔和渐变 |
| 阴影风格 | 柔和分层阴影 + 电影感氛围光 | 面部 2 层阴影（主阴 + 反射光）；环境用全局光照 |

## 二次元化程度

**5/10 — 写实基底 + 二次元精致感**

介于《原神》（偏 7-8 分动漫）与写实 CG（偏 1-2 分）之间，接近《逆水寒》CG 的质感：
- 头身比：成年角色 7 头身，非 Q 版
- 脸型：五官比例接近真人但稍精致化（大眼、高鼻梁、小嘴）
- 表情：克制写实，不夸张到日系萌系的程度
- 肌肤：有光泽和质感，无毛孔和皱纹细节
- 服饰：面料质感清晰（丝绸反光、刺绣立体），但非影视级布料模拟

## 朝代设定：架空古风

**不对应具体朝代**，视觉元素可混搭：

- **建筑**：以汉/唐/宋风格为主视觉骨架（朱红宫墙、飞檐翘角、汉白玉栏杆、窗棂雕花、瓦当纹样）
- **服饰**：融合唐制（高腰齐胸襦裙、圆领袍）、宋制（褙子、对襟衫）、少量明制元素（立领、马面裙），不强制考据
- **色彩基调**：朱红、靛青、石青、金、雪白；敦煌/宋画的内敛色调，避免纯饱和原色
- **道具纹样**：云纹、回纹、缠枝、牡丹、鹿纹、竹叶

## 标准 Prompt 片段（强制复用）

后续阶段（02 Moodboard、04 场景库、05 角色设计、07 视频）必须在 prompt 中包含以下关键词组合（可按场景调整侧重）：

### 核心关键词（必用）

```
3D渲染，CG动画风格，游戏原画，国风3D动漫，二次元
```

### 语言规范（强制）

> **所有生成 prompt 必须使用纯中文**。禁止在 prompt 中混入英文术语。
> 技术性渲染术语统一使用以下中文等价词：

| 英文术语（禁用） | 中文等价词（必用） |
|----------------|-----------------|
| PBR material | 物理渲染材质 |
| cinematic lighting | 电影级光影 |
| volumetric fog | 体积雾效 |
| volumetric light | 体积光 |
| volumetric atmosphere | 体积大气 |
| subsurface scattering | 次表面散射 |
| delicate details | 精致细节 |
| depth of field | 景深虚化 |
| movie-grade composition | 电影级构图 |
| fantasy Chinese palace | 幻想风中式宫殿 |
| golden hour | 黄金时刻 |
| silhouette | 剪影 |
| traditional embroidery | 传统刺绣 |
| silk texture | 丝绸质感 |
| Chinese ancient architecture | 中式古代建筑 |
| detailed eye highlights | 精致眼部高光 |
| flowing hair strands | 飘逸发丝 |
| delicate anime face | 精致动漫面部 |
| imperial court | 皇家宫廷 |

> **理由**：Seedream 5.0 对纯中文 prompt 理解能力充分，混合语言会导致 prompt 风格不统一、团队协作时阅读困难。统一纯中文降低沟通成本。

### 扩展关键词（按需选用）

| 类型 | 关键词 | 使用时机 |
|------|--------|---------|
| 质感强化 | 物理渲染材质，电影级光影，次表面散射 | 强调材质和光影 |
| 画面感 | 电影级构图，景深虚化，体积光 | 镜头语言明显的镜头 |
| 国风细化 | 中式古代建筑，传统刺绣，丝绸质感 | 场景/服饰特写 |
| 二次元精致 | 精致动漫面部，精致眼部高光，飘逸发丝 | 人物面部特写 |
| 架空氛围 | 幻想风中式宫殿，皇家宫廷 | 宫廷场景 |

### 避免关键词（禁用）

- ❌ `photorealistic` / `hyperrealistic`（写实——会把二次元感拉没）
- ❌ `cell shaded` / `toon shader`（卡通渲染——会变成硬描边）
- ❌ `chibi` / `Q version`（萌系头身比）
- ❌ `western cartoon` / `American comic`（美漫风）
- ❌ 任何英文术语混入中文 prompt（统一纯中文，见上方语言规范）

## 避免的风格

- ❌ 写实 CG（《阿凡达》《长安三万里》级真实感）
- ❌ 纯 2D 赛璐珞（《间谍过家家》硬描边平涂）
- ❌ 美漫 3D（《蜘蛛侠：平行宇宙》网点变形）
- ❌ Q 版萌系（2-3 头身）

## 与后续阶段的约束

| 阶段 | 约束 |
|------|------|
| 02 Moodboard | 5-8 张风格图必须符合"三渲二国风 + 架空"定位 |
| 04 场景库 | Seedream prompt 强制包含"核心关键词" + 场景类扩展词 |
| 05 角色设计 | Seedream prompt 强制包含"核心关键词" + 二次元精致扩展词 |
| 07 视频生成 | Seedance prompt 在场景/角色 prompt 基础上继承视觉风格，保持一致性 |

## 一句话总结

**"国风版《原神》过场 CG 的质感，但偏写实二次元（5/10 动漫度），架空朝代不考究。"**
