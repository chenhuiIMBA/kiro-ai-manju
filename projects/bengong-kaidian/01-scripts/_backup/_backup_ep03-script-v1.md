# 第 3 集：臣妾想回家

**时长**：120s
**结构模板**：B（紧凑单场景为主）
**集主题推进**：萧珩第一次笑（B线关键伏笔）+ 温婉善意首次释放 + 柳如烟开始警觉
**故事线推进**：A线:作死误读持续（"想回家"被解读为"率真"）| B线:萧珩第一次笑 | C线:温婉友谊萌芽 | D线:柳如烟启动调查

---

## 时间轴

【0-3s，共 3s】钩子（反差型）
画面：大殿内众嫔妃整齐跪拜，动作优雅统一。镜头从队列中缓缓推进，停在一个跪姿明显歪斜的身影上——沈鹿溪跪错了方向，面朝侧面。画面边缘殿门口，小桃探头急得直跺脚。
镜头语义：远景（整齐跪拜）→ 推进至中景（沈鹿溪跪歪，殿门口小桃跺脚）
音频：<衣料摩擦声 + 小桃急促的跺脚声>

【3-30s，共 27s】场景 1（选秀大殿 — 集体请安）
画面：大殿内，嫔妃们跪拜完毕起身。萧珩坐在高台上，面无表情俯视。太监宣布"皇上有旨，新入宫嫔妃逐一上前请安"。
镜头语义：中远景（大殿全貌）→ 中景（萧珩高台）

画面：沈鹿溪独自走到殿中央，手忙脚乱整理衣袖，差点被自己裙摆绊倒。
镜头语义：中景（沈鹿溪走出）→ 中近景（手忙脚乱）

小桃（殿门口方向传来急促低声）："小姐！左脚先迈！左脚！"
> emotion: nervous, speed: 0.2, loudness: -0.2

画面：沈鹿溪站定，深吸一口气，抬头直视高台上的萧珩。全场安静。她鼓起勇气开口。
镜头语义：中近景（沈鹿溪抬头）→ 中远景（她与萧珩的距离感）

沈鹿溪（鼓起勇气，声音清晰）："臣妾……想回家。"
> 潜台词：她是认真的——这是她唯一想说的话
> emotion: neutral, speed: -0.1, loudness: 0

画面：全场死寂。嫔妃们倒吸一口凉气。殿门口小桃探头双手捂脸。
镜头语义：中远景（全场反应）→ 中景（殿门口小桃捂脸）

【30-65s，共 35s】场景 1 续（选秀大殿 — 萧珩第一次笑）
画面：萧珩坐在高台上，沉默。右手拇指摩挲玉佩，节奏比平时慢。全殿无人敢出声。
镜头语义：中近景（萧珩沉默）→ 特写（拇指摩挲玉佩）

画面：漫长的三秒沉默后，萧珩嘴角微微动了一下——不是笑，但比不笑多了一丝弧度。这是他第一次笑。
镜头语义：特写（萧珩嘴角微动）

萧珩（语气平淡，但比平时轻了半分）："朕知道了。"
> 潜台词：三个字，但语气不是拒绝——他觉得有趣
> emotion: neutral, speed: -0.2, loudness: -0.1

画面：沈鹿溪愣住——她预期的是被骂或被罚，没想到是这个反应。全场嫔妃面面相觑。
镜头语义：中近景（沈鹿溪愣住）

画面：请安散后，嫔妃们三三两两离开。沈鹿溪站在原地发呆。一个穿淡紫色衣裙的身影走过来——温婉，笑眼弯弯，手中端着一碟桂花糕。
镜头语义：中景（嫔妃散去）→ 中近景（温婉走来）

温婉（轻声，笑眼弯弯，递出桂花糕）："妹妹别怕。尝尝这个，我自己做的。"
> 潜台词：她是唯一主动释放善意的人——但她的善意背后有太后的任务
> emotion: tender, speed: -0.1, loudness: -0.1

沈鹿溪（接过桂花糕，有点懵）："……谢谢。"
> emotion: nervous, speed: 0, loudness: -0.2

【65-95s，共 30s】场景 1 续（选秀大殿 — 温婉+沈鹿溪）
画面：温婉和沈鹿溪并肩站在殿外廊下，沈鹿溪咬了一口桂花糕，眼睛微微亮了。
镜头语义：中景（二人廊下）→ 中近景（沈鹿溪咬桂花糕）

沈鹿溪（嚼着桂花糕，认真评价）："桂花是新鲜的，糖渍时间刚好……但糯米粉可以再细一点。"
> 潜台词：食肆世家本能——她在评价食物，也在放下戒备
> emotion: happy, speed: 0, loudness: 0

温婉（捂嘴轻笑）："妹妹懂吃。下次我改。"
> emotion: happy, speed: 0, loudness: -0.1

画面：小桃从柱子后面探出头，看到沈鹿溪在笑，松了一口气，自己也偷偷笑了。
镜头语义：中景（小桃探头偷看）→ 中近景（小桃偷笑）

【95-120s，共 25s】场景 2（锦华宫 — 柳如烟警觉）
画面：锦华宫内，柳如烟坐在梳妆台前，铜镜映出她精致的面容。侍女在旁汇报今日请安的情况。
镜头语义：中景（柳如烟梳妆台前）→ 特写（铜镜中的面容）

画面：柳如烟听到"皇上笑了"时，手中团扇"啪"一声合上。铜镜中她的眼神骤然变冷。
镜头语义：特写（团扇合上）→ 中近景（眼神变冷）

柳如烟（语气平静但眼神冰冷）："皇上……笑了？"
> 潜台词：萧珩从不笑——这个沈氏凭什么
> emotion: coldness, speed: -0.2, loudness: 0

画面：柳如烟站起身，走到窗前，背对侍女。
镜头语义：中景（柳如烟走向窗前）

柳如烟（背对侍女，语气不紧不慢）："去查查这个沈氏的底细。家世、入宫前做什么的……事无巨细。"
> 潜台词：她嗅到了威胁
> emotion: coldness, speed: -0.1, loudness: -0.1

画面：窗外宫墙在夕阳中投下长影，柳如烟的侧脸半明半暗。
镜头语义：中近景（柳如烟侧脸半明半暗）→ 画面渐暗
— 悬念：柳贵妃下令"查查这个沈氏的底细"
