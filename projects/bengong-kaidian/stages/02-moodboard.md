# 阶段 02：风格板（Moodboard）

> 工具：Seedream 5.0（图片生成）
> 前置：`ARK_API_KEY` 已设置，`video-generation` skill 可用

## 前置条件

- `01-scripts/series-bible.md` 已确认
- `01-scripts/visual-style-spec.md` 存在

## 流程

### 步骤 1：选定场景类型覆盖 + 变体轮换

> 组装 prompt 前先确定本轮要覆盖哪些场景类型和变体，避免跑出 5 张相似图。

#### 1a：选取场景类型（从下方池子选 5-8 种）

> ⚠️ **Moodboard 禁止包含具体角色形象**。角色设计在阶段 05，Moodboard 阶段角色尚未确认。如果概念图中出现人物，只能是模糊剪影、背影、或无法辨识面部的远景人影——不能出现具体的面部特征、发型、服装细节。

| 类型 | 示例 prompt | 灯光要点 |
|------|------------|---------|
| 白天外景 | `午后阳光透过树叶洒落地面，暖黄色调，温暖日常氛围` | 顶光/侧光，光比 2:1~3:1，暖色温 |
| 夜间内景 | `昏暗室内，窗外月光照入，冷蓝色调，神秘安静氛围` | 月光主光源（冷），可能有暖色人工光点缀 |
| 逆光/剪影 | `角色背对落日，身体剪影被金色光线勾勒轮廓，天空橙红渐变` | 主体曝光不足，背景高亮，高反差 |
| 室内混合光源 | `咖啡厅靠窗，窗外阴天漫射冷光，桌面暖黄台灯补光` | 冷暖色温共存 |
| 大场景氛围 | `俯瞰城市夜景，万家灯火，深紫与金色交织` | 无单一主光，靠整体层次 |
| 黄金时刻 | `日落前30分钟，整个场景被金色暖光包裹，影子被拉长` | 低角度暖光，长阴影，色温 ~3500K |
| 蓝调时刻 | `日落后20分钟，天空深邃蓝色，地面灯光点点，冷暖强烈对比` | 天空冷蓝为主，地面暖光点缀 |
| 雨景/水景 | `雨中街道，地面湿漉反射霓虹灯光，雨滴在灯光中闪烁` | 多次反射/散射光源，高光点密集 |

根据题材选 5-8 种，覆盖至少 3 种不同光影类型。

#### 1b：变体轮换（三池各取 1-2 种）

> 同一个"日系动漫风格概念插画"跑 5 次会产出高度相似的光影和构图。从以下三个池子**主动注入变体描述**，至少各取 1 种，建议各取 2 种。

**色彩策略池**（至少取 3 种轮换）：

| 色彩策略 | 描述短语 | 情绪 |
|---------|---------|------|
| 暖金主调 | "暖金色阳光包裹" | 温暖、希望、日常 |
| 冷蓝主调 | "冷蓝色调笼罩" | 孤独、理性、深夜 |
| 冷暖对冲 | "暖橙肤色与冷蓝背景形成强烈对比" | 内心挣扎、对峙 |
| 低饱和沉郁 | "去饱和的灰调，色彩被抽离" | 失落、回忆、压抑 |
| 高饱和鲜艳 | "色彩鲜明饱和，如插画般绚丽" | 幻想、青春、活力 |
| 单色渐变 | "单一色相渐变，从深蓝到浅蓝的层次" | 高度风格化、梦境 |

**构图表征池**（至少取 3 种轮换）：

| 构图策略 | 描述短语 | 适合 |
|---------|---------|------|
| 对称中央 | "对称构图，主体位于画面正中央" | 庄严、压迫感 |
| 三分之一偏移 | "主体偏移至右侧1/3，左侧留白" | 孤寂、思考、看向远方 |
| 极低角度 | "极低角度仰拍，主体占据画面上方2/3" | 强大、威胁、仰视感 |
| 俯瞰扁平 | "俯瞰视角，主体只是场景中的一个小点" | 渺小、宿命、全景叙事 |
| 前景遮挡 | "透过前景物体（门框/树叶/窗格）偷窥式构图" | 窥视、秘密、疏离 |
| 极端特写 | "只取局部特写，背景全虚" | 情感爆发、内心戏 |

**光影氛围池**（可选 1-2 种）：

| 氛围 | 描述短语 | 适用题材 |
|------|---------|---------|
| 晨雾 | "薄雾弥漫，光线柔和散射，空气感强" | 清新、悬疑 |
| 正午烈日 | "强烈顶光，阴影锐利短小，高反差" | 紧张、对峙 |
| 烛光/火光 | "单一烛光从下方照亮，周围全暗" | 秘密、亲密、恐怖 |
| 霓虹夜色 | "霓虹灯光在地面水渍中反射，多色污染" | 都市、赛博 |
| 阴天漫射 | "阴天均匀漫射光，无明确阴影，灰冷色调" | 压抑、日常、厌倦 |
| 闪电瞬间 | "闪电瞬间的白光照亮整个场景，影子极长" | 惊悚、转折 |

#### 1c：组合表格

本轮选定的组合写入临时表格，后续组装 prompt 时引用：

| 图片编号 | 场景类型 | 色彩策略 | 构图策略 | 光影氛围（可选） |
|---------|---------|---------|---------|----------------|
| mood_01 | 夜间内景 | 冷蓝主调 | 前景遮挡 | 烛光 |

| mood_03 | 蓝调时刻 | 单色渐变 | 对称中央 | 霓虹夜色 |
| ... | ... | ... | ... | ... |

### 步骤 2：组装 prompts 并批次审核

对照 `references/prompt-guide.md` §八「图片 Prompt 审查清单」逐张自查。

**Prompt 模板**（引用步骤 1c 表格的组合）：

```
{画风}风格概念插画。{场景类型描述}，{色彩策略描述}，{构图策略描述}，{光影氛围描述}。高品质概念插画。
```

> ⚠️ **Prompt 审核门禁（门禁 1）**：所有图片的 prompt 组装完成后，一次性批次发送给用户审核，不逐张发。确认后进入步骤 3。

### 步骤 3：生成 + 即时登记

**每张图执行三连**（即时登记门禁 2）：

```bash
# 1. 生成
python3 <video-gen>/scripts/seedream.py create \
  -p "<prompt>" \
  --size 9:16 \
  -d ./02-moodboard/

# 2. 重命名
mv ./02-moodboard/seedream_*.jpg ./02-moodboard/mood_{NN}.png

# 3. 立即登记 assets.md（见步骤 4 模板）
```

> ⚠️ 禁止批量生成 5 张后再一起补登记。必须一张一登记。

提交 5-8 张图 + 变体记录给用户确认——"就这个感觉"。不满意则调整变体组合重新跑，**确认后再进灯光哲学**。
